# fb
It´s showtime!

## Installation
With [node](nodejs.org) installed:

````
npm i && bower install && npm i -g gulp && gulp
````

## Debugging
`DEBUG=calendar:* gulp`

## Directory structure
````
.
├── build          // Compiled files
├── client         // Source files
│   ├── fonts
│   ├── images
│   ├── modules
│   ├── styles
│   └── vendor     // Bower packages
├── node_modules   // Node packages
└── test
````
